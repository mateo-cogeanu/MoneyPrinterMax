# Project Goals

This project should evolve toward making MoneyPrinterTurbo more productive, more automated, and easier to rely on in day-to-day use.

Primary goals:

- Improve productivity for users by reducing manual steps, repeated setup work, and avoidable friction in common workflows.
- Increase useful automation across the app while keeping behavior predictable and easy to inspect.
- Identify and fix quirks, rough edges, and confusing behaviors as they are reported.
- Prefer focused, practical changes that make the app feel smoother without creating unnecessary complexity.

When new quirks or workflow issues are described, treat them as actionable improvement items: understand the current behavior, make the smallest reliable fix, and verify the result before moving on.
